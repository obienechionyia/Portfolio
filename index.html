My Projects

  • Penny Flights

    This is an application that allows users to enter their location, and receive links to the cheapest flights available from the entered origin, as well as the cheapest hotels in each destination. The Skyscanner and IPify APIs were used in order to pull the necessary data and apply it the application. Tech used: Javascript, React.js, HTML, and CSS.

  • BestFlix

    I created an application that shows the top movies and shows from the IMDB API and displays them for the user in a format similar to Netflix's. The user can then choose any film or show on display and find a link to a trailer, or add it to a "Watch Later" list. Tech used: Javascript, React.js, HTML, and CSS.
