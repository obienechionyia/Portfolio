Web Dev Projects

  • Penny Flights

    This is an application that allows users to enter their location, and receive links to the cheapest flights available from the entered origin, as well as the cheapest hotels in each destination. The Skyscanner and IPify APIs were used in order to pull the necessary data and apply it the application. Tech used: Javascript, React.js, HTML, and CSS

  • BestFlix

    I created an application that shows the top movies and shows from the TMDB API and displays them for the user in a format similar to Netflix's. The user can then choose any film or show on display and find a link to a trailer, or add it to a "Watch Later" list. Tech used: Javascript, React.js, HTML, and CSS

  • Landing Page

    This is a basic website landing page for a fake company, made with HTML/CSS/Bootstrap and Javascript. Tech used: Javascript, React.js, HTML, and CSS

  • Nudge

    This is a front-end app which allows users to add, edit, and delete tasks from a custom list. It features a simple, easy to use interface that allows for easy editing and updates. Tech used: Javascript, HTML, and CSS

  • Weather App

    This app uses the OpenWeatherMap API to retrieve current weather forecasts in any city or zip code in the world, and display them to the user either in the terminal or through a GUI made with tkinter. It also uses geolocation tracking to allow users to retrieve weather info for their current location. Tech used: Python, Tkinter
