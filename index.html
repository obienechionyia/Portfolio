  • Penny Flights

    As a travel enthusiast, I embarked on crafting Penny Flights to empower wanderers to explore without breaking the bank. I designed an intuitive interface using React that ensures effortless flight searches and bookings. The most challenging part of this project was dealing with various bugs that were caused by API changes. I learned a lot about the patience it takes to continuously support an application, and my ability to identify bugs in a timely manner improved greatly.

  • BestFlix

    BestFlix is a React-based Netflix clone that uses the TMDB API to offer a wide variety of movies and TV shows, including original content. It also features an interactive "Watch Later" list, so you can easily keep track of what you want to watch. I used this project to practice my CSS skills, which improved considerably as a result.

  • Nudge

    Nudge is a JavaScript-based CRUD reminder application that helps you stay organized and on top of your tasks. It allows you to create reminders, edit them, and delete them as needed. Although simple, this app displays the importance of straightforwardness in a CRUD app, with an incredibly user-friendly interface.

  • Hiking Landing Page

    I created an interactive, photo-based sample landing page for a hiking website. I was given a design that was created in Photoshop by the client, and was asked to recreate the design using HTML, CSS and Javascript, which I was able to do effectively. The most challenging part of this project was sticking to the script; although I wanted to add my own ideas to the project, I understood the importance of creating the site exactly how the client wanted it.

  • Weather App

    I created a weather app that shows the current weather status in any city in the world. The app is written in Python and uses the OpenWeatherMap API to get weather data. This is one of my favorite applications, as it was the first functional project I built during my development journey. With this project, I learned how to connect a back-end API database to to a front-end application.
